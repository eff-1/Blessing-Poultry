# LOGO & BRANDING IMAGES

Put your logo and branding files here.

## Required Files:

1. **logo.png** - Main logo with transparent background (500x500px)
2. **logo-white.png** - White version of logo for dark backgrounds (500x500px)
3. **favicon.ico** - Small icon for browser tab (32x32px and 16x16px)

## Optional:
4. **logo-text.png** - Logo with text/farm name
5. **logo-icon.png** - Just the icon part of logo

## Image Specs:

### Main Logo (logo.png):
- Size: 500x500px
- Format: PNG with transparent background
- Quality: High resolution, vector-based if possible
- File size: Under 100KB

### Favicon (favicon.ico):
- Size: 32x32px and 16x16px (multi-size ICO file)
- Format: ICO
- Simple, recognizable at small size

## Tips:
- Logo should be clear and professional
- Works on both light and dark backgrounds
- Simple enough to be recognizable when small
- Represents your poultry farm brand
- If you don't have a logo, we can use text-based branding
